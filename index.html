How to upgrade from older Blackcoin wallets?

From the Blackcoin original wallet v1.2.x.

As the original wallet isn't able to synchronize with the blockchain, sending funds from the original wallet to your new More wallet isn't an option anymore. Follow this link for instructions on how to dump and import the wallet.

From the Blackcoin Lore wallet v2.12.x

Technically you can point Blackcoin More installation to your data folder, but it is recommended to create a new installation in the default data folder. The same for your funds, it is technically possible to use the same wallet.dat file, but it is better to use the wallet created by new installation. Send your funds to this new wallet.

From the Blackcoin More wallet v2.13.x

You can upgrade just by replacing binaries, but backup your wallet first (wallet.dat)

My balance is incorrectly displayed, what should I do?

No worries, in Blackcoin More v2.13.2.9 we most likely fixed this, as the LevelDB component was upgraded.
